# Store API

Aplicação front-end desenvolvida com React e Vite para consumo de API REST de uma loja virtual.

## 🚀 Tecnologias

- React
- Vite
- JavaScript
- CSS Modules
- HTML5
- Fetch API
- React Router DOM

## 📌 Funcionalidades

- Listagem de produtos
- Visualização de detalhes de um produto
- Consumo de uma API REST fake utilizando Fetch API.
- Navegação entre páginas com rotas
- Gerenciamento de estado no React
- Adição de produtos ao carrinho

## 📚 Aprendizados

Este projeto foi desenvolvido com o objetivo de praticar a criação de aplicações em React, consumo de APIs REST, roteamento entre páginas e gerenciamento de estado.

## ▶️ Como executar

```bash
# Clone o repositório
git clone https://github.com/Manoel7Oliveira/Store-Api.git

# Acesse a pasta
cd Store-Api

# Instale as dependências
npm install

# Execute o projeto
npm run dev
```
